Pin Descriptions

PWMC

The PWM generator needs to run at a higher clock speed than the SID itself (typ. 1MHz), and as such, needs its own clock source. This is typically a 16 - 20MHz clock. This pin MUST NOT be connected directly to Vcc or GND. If PWM is unused, tie this pin to the main clock source.

PWM

The PWM audio out is provided here. This signal has a duty cycle (sample) resolution of 9-bit, on a carrier wave of ~31KHz (given the minimum 16MHz clock on PWMC).

D0 - D7

These bi-directional pins, together with A0 - A4, R/W̅ and C̅S̅ form a 8-bit parallel data bus, which is compatible with that of the 6581/8580 SID. Data is transfered in and out of the device using these pins.

A0 - A4

5-bit address bus used to select one of the internal registers of the SID.

R/W̅

Selects between read and write operations. On read, data from the selected register becomes available on the data buss. On write, the selected register will be loaded with the data provided on the data bus.

C̅S̅

Chip Select input. If high, D0 - D7 are held in a high-impedance state, and the state of R/W̅ will be ignored.

S0 - S15

These outputs update on every positive going clock edge with the current, mixed sample of all SIDs. Each 16-bit sample is provided in parallel on these pins, and may read by an external DAC, or other digital device.

DAC_CE

Chip-select line for a SPI DAC externally connected to the AS6581. This signal is either active high or active low, depending on the current SPI mode.

DAC_SCLK

Serial Clock for a SPI DAC externally connected to the AS6581. Serial data is to be latched on either the rising or falling edge of this signal, depending on the current SPI mode.

DACM_SI

Serial Data In to a SPI DAC externally connected to the AS6581. Serial data is either 12- or 16-bit depending on the current SPI mode, and contains the mixed sample from all SIDs.

DAC0_SI - DAC7_SI

Serial Data In to a number of SPI DACs externally connected to the AS6581. Serial data is either 12- or 16-bit depending on the current SPI mode, and contains the current sample of one of the individual SIDs.
