Register Description

Notice: This documentation does not serve to repeat any information on the original SID architecture. Please see the original 6581 datasheet for this information.

This page mostly serves to document any registers specific to the AS6581, and to highlight differences between the MOS 6581 and AS6581.

All registers are read/write unless otherwise specified (even those designated write-only in the MOS 6581’s datasheet).

Standard SID Registers

Voice 1 Registers

Freq Lo

Base address: 0x00

Low byte of Voice 1 frequency setting.

Freq Hi

Base address: 0x01

High byte of Voice 1 frequency setting.

PW Lo

Base address: 0x02

Low byte of Voice 1 pulse-width setting.

PW Hi

Base address: 0x03

High nibble of Voice 1 pulse-width setting.

Control Reg

Base address: 0x04

Control reg for Voice 1. Each bit controls a setting for Voice 1.

Attack/Decay

Base address: 0x05

Attack and Decay time settings for Voice 1.

Sustain/Release

Base address: 0x06

Sustain level and Release time settings for Voice 1.

Voice 2 Registers

Freq Lo

Base address: 0x07

Low byte of Voice 2 frequency setting.

Freq Hi

Base address: 0x08

High byte of Voice 2 frequency setting.

PW Lo

Base address: 0x09

Low byte of Voice 2 pulse-width setting.

PW Hi

Base address: 0x0A

High nibble of Voice 2 pulse-width setting.

Control Reg

Base address: 0x0B

Control reg for Voice 2. Each bit controls a setting for Voice 2.

Attack/Decay

Base address: 0x0C

Attack and Decay time settings for Voice 2.

Sustain/Release

Base address: 0x0D

Sustain level and Release time settings for Voice 2.

Voice 3 Registers

Freq Lo

Base address: 0x0E

Low byte of Voice 3 frequency setting.

Freq Hi

Base address: 0x0F

High byte of Voice 3 frequency setting.

PW Lo

Base address: 0x10

Low byte of Voice 3 pulse-width setting.

PW Hi

Base address: 0x11

High nibble of Voice 3 pulse-width setting.

Control Reg

Base address: 0x12

Control reg for Voice 3. Each bit controls a setting for Voice 3.

Attack/Decay

Base address: 0x13

Attack and Decay time settings for Voice 3.

Sustain/Release

Base address: 0x14

Sustain level and Release time settings for Voice 3.

Filter registers

FC Lo

Base address: 0x15

Lowest 3 bits for filter cutoff frequency setting.

FC Hi

Base address: 0x16

Highest 8 bits for filter cutoff frequency setting.

RES/Filt

Base address: 0x17

Filter resonance setting, and bit-mask for which channels should use the filter. Filt EX is unimplemented in the AS6581, and setting it will have no effect.

Mode/Vol

Base address: 0x18

Filter mode setting and volume setting.

Misc registers

POTX

Base address: 0x19 (read-only)

Notice: This register is not implemented in the AS6581, and will always read 0.

POTY

Base address: 0x1A (read-only)

Notice: This register is not implemented in the AS6581, and will always read 0.

OSC3/Random

Base address: 0x1B (read-only)

Returns the highest 8 bits of the current value output by Voice 3’s tone generator.

ENV3

Base address: 0x1C (read-only)

Returns the current envelope volume of Voice 3.

SID Fine-Tune registers

These registers use addresses originally unused by the SID, to include registers meant to fine-tune the sound of the SID post-production. They are meant to compensate for the uncertainty about the sound generated by the real hardware, as simulations of the chip run in software are not 100% accurate. The following registers are not meant to be modified by the user, instead being loaded with manually fine-tuned values by the management controller on power-up or reset.

Filter offset hi

Base address: 0x1D

Filter offset lo

Base address: 0x1E

The filter offset is a 14-bit value added onto the raw output of the SID filter, to compensate for signal level differences. Its loaded with the default value of 8192 on power-up or reset. It also has the effect of pushing the filter output entirely into the positive number range. If filter offset is set to a lower value, the filter will begin producing negative numbers, which would be clamped to 0.

Filter amplitude adjust

Base address: 0x1F

The raw filter output will be right-shifted (exponentially reduced in amplitude) according to the setting in this register. Initialized to 0 on power-up or reset, and most likely unused, unless the characteristics of the physical chip end up differing greatly from expectations.

Global Configuration Register

This is probably the most important register in the AS6581, as it is what controls the various output modes, mixer modes and contains the SID select bits.

Base address: 0x19 (write-only)

Information about the functionality controlled by this register is documented on their own pages.

For information on SIDSEL, see Addressing multiple SIDs.

For information on MMODE, see Mixer configuration.

For information on SPI_mode, SPI_rst and SPI_en, see SPI DAC output mode.

This register is global, meaning there exists only one copy of it, and writing to this address will always write this same register, no matter the value of SIDSEL.

SID Mask Register

Base address: 0x1A (write-only)

This register contains an 8-bit mask of which SIDs are currently enabled. Outputs of SIDs for which the bit is 0 in this register will not be mixed into the final output.
